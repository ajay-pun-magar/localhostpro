# Localhost pro
Localhostpro is a platform built on Python fastAPI where you can create your personal HTML page and share online.
There are many too already which are very advanced but Localhostpro is intended to be very simple and beginner friendly.
The most discouraging and diffcult problem among non technical people and beginnners is getting your HTML page online. 
It's either stuck on your directory as a .html file or localhost server and never goes out online on the internet.
Localhostpro solves this exact problem by allowing you to edit you HTML page and host it on a single CLICK.

# How to run?
**Note:** _Docker is required._
Run the terminal inside **localhostpro** directory.
Run command: **docker-compose up --build -d**
Open "http://localhost:8000/" in your browser.
